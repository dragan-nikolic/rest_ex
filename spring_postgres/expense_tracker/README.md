## Setup

* Java JDK 8
* IntelliJ IDE
* Postman
* Docker
* PostgreSQL

### PostgreSQL

Use docker [image](https://hub.docker.com/_/postgres).

Run: `$ docker container run --name posgresdb -e POSTGRES_PASSWORD=admin -d -p 5432:5432 postgres`
Check: `$ docker container ls`
